# Content Atomizer — Output Folder

## What Appears Here

```
output/
└── atoms/
    ├── twitter-thread.md     # Ready-to-post thread
    ├── linkedin-post.md      # Professional post
    ├── newsletter-section.md # Email segment
    ├── quote-cards.md        # Visual quotes
    └── video-clips.md        # Clip suggestions
```

---

## File Descriptions

### twitter-thread.md
Complete thread including:
- Numbered tweets
- Hook analysis
- Hashtag suggestions
- Posting timing

### linkedin-post.md
Platform-ready post:
- Under character limit
- Hook in first 2 lines
- Professional format
- Engagement strategy

### newsletter-section.md
Email-ready content:
- Standalone value
- Link to full content
- Subject line suggestion

### quote-cards.md
Visual-ready quotes:
- Extracted standout lines
- Platform suggestions
- Design notes

### video-clips.md
Clip recommendations:
- Timestamps for best moments
- ffmpeg extraction commands
- Platform targeting

---

## Using the Outputs

### Publishing Workflow
1. Review each atom for accuracy
2. Customize as needed (add emojis, adjust tone)
3. Schedule or post
4. Engage with responses

### Scheduling Suggestion
Don't post all atoms same day:
- Day 1: Twitter thread
- Day 2: LinkedIn post
- Day 3-5: Quote cards
- Newsletter: Next send

### Tracking
Note what performs best to inform future atomization.
