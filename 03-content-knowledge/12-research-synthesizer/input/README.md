# Research Synthesizer — Input Folder

## What Goes Here

Documents and sources to include in synthesis.

### Supported Documents

| Format | Best For |
|--------|----------|
| `.pdf` | Research papers, reports |
| `.docx` | Business documents |
| `.txt` / `.md` | Articles, notes |
| `.html` | Saved web pages |

---

## Folder Structure

```
input/
├── primary-sources/      # Most important documents
├── secondary-sources/    # Supporting materials
├── research-config.md    # Optional: parameters
└── ignore/               # Skip these files
```

---

## Research Configuration

```markdown
# research-config.md

## Research Question
[What you want to understand]

## Focus Areas
- [Area 1]
- [Area 2]

## Source Preferences
- Prioritize: Academic, recent
- Include: Industry reports
- Avoid: Marketing materials

## Output Preferences
- Brief length: Comprehensive / Overview
- Citation style: Footnotes
```

---

## Without Input Files

Web-only research works through conversation:
> "Research [topic] from web sources"

---

## Tips

1. **Quality over quantity**: Few good sources beat many mediocre
2. **Include key documents**: If you have authoritative sources, add them
3. **Note conflicts**: "Sources A and B disagree on X"
4. **Specify focus**: Narrower question = better synthesis
