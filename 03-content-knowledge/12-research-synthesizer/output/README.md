# Research Synthesizer — Output Folder

## What Appears Here

```
output/
├── research-brief.md      # Executive summary
├── source-comparison.md   # Agreement/disagreement
├── findings.md            # Detailed breakdown
└── sources/
    └── [organized source notes]
```

---

## File Descriptions

### research-brief.md
Executive summary including:
- Key findings with citations
- Synthesized conclusions
- Recommendations
- Full reference list

### source-comparison.md
Source analysis:
- Source credibility ratings
- Agreement matrix
- Conflict analysis
- Trust recommendations

### findings.md
Detailed breakdown:
- Each finding expanded
- Supporting evidence
- Implications

### sources/
Organized source materials:
- Extracted key quotes
- Source summaries
- Reference details

---

## Using the Outputs

### For Quick Understanding
Start with `research-brief.md` executive summary.

### For Detailed Analysis
Review `findings.md` and check citations.

### For Source Trust
Consult `source-comparison.md` before relying on specific claims.

### For Further Research
Use `sources/` to trace back to originals.
