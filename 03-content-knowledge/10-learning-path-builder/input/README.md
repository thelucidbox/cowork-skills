## What Goes Here

Optional context files to personalize your learning path.

### Configuration Files

| File | Purpose |
|------|---------|
| `learning-profile.md` | Your preferences and constraints |
| `current-knowledge.md` | What you already know |
| `goals.md` | Why you're learning this |

### Previous Paths

| Folder | Purpose |
|--------|---------|
| `completed/` | Finished learning paths for reference |
| `in-progress/` | Current paths being worked on |

---

## Learning Profile Example

```markdown
# learning-profile.md

## Time Available
- Hours per week: 5
- Best learning time: Mornings
- Deadline: None / By [date]

## Learning Style
- Prefer: Video tutorials, hands-on projects
- Avoid: Dense textbooks
- Best format: Short lessons (< 30 min)

## Constraints
- Budget: Free only / Up to $X
- Platform access: LinkedIn Learning, O'Reilly

## Background
- Related skills: [What you know]
- Learning history: [Past relevant learning]
```

## Tips

1. **Be honest about level**: Better path if we know truth
2. **Share constraints**: Budget, time, style all matter
3. **Include goals**: "For work" vs "for fun" shapes path
4. **Note failed attempts**: Helps avoid same mistakes
