## Curriculum Design Rules

### Phase Duration
- No phase longer than 4 weeks
- Prefer 1-2 week phases for momentum
- Include built-in milestones

### Difficulty Progression
- Start slightly below assessed level (confidence building)
- Gradual increase
- Each phase builds on previous
- Practical application in each phase

### Resource Selection
- Prefer free over paid (note when paid is worth it)
- Mix formats (video + reading + practice)
- Current resources for fast-changing topics
- Classics for fundamentals

---

## First Assignment Rules

### Characteristics
- Completable in 1-2 hours max
- Immediate tangible result
- Builds confidence
- Relevant to end goal
- Doesn't require extensive setup

### Don't
- Start with theory
- Require lots of installation
- Take more than one session
- Feel like homework

---

## Quality Checklist

- [ ] Path matches stated goal
- [ ] Phases are appropriate length
- [ ] Resources match learner level
- [ ] Mix of resource types
- [ ] Free resources prioritized
- [ ] First assignment is immediately actionable
- [ ] Milestones are verifiable
- [ ] End goal is clearly achievable via path
