# Learning Path Builder — Output Folder

## What Appears Here

```
output/
├── learning-path.md       # Complete curriculum
├── resources.md           # Curated resources
├── first-assignment.md    # Start today
└── progress-tracker.md    # Track advancement
```

---

## File Descriptions

### learning-path.md
Complete curriculum including:
- Phases with duration
- Topics per phase
- Milestones to verify progress
- Final project

### resources.md
Curated learning materials:
- Primary resources (start here)
- Secondary resources (deep dives)
- Community and practice platforms
- Reference materials

### first-assignment.md
Immediate starting task:
- Clear steps
- Achievable in 1-2 hours
- Tangible result
- Confidence building

### progress-tracker.md
Track your journey:
- Phase completion checkboxes
- Weekly log template
- Reflection space
- Celebration tracker

---

## Using the Outputs

### Getting Started
1. Review `learning-path.md` for overview
2. Check `resources.md` for where to learn
3. Start with `first-assignment.md` today
4. Set up `progress-tracker.md`

### Ongoing
- Update tracker weekly
- Check off milestones
- Adjust if phases take longer/shorter
- Add notes for future reference

### Completion
- Verify against success criteria
- Archive in completed paths
- Ask for next learning path

---

## Adjusting the Path

If the path isn't working:
> "Phase 2 is taking longer than expected"
> "I want to skip to [topic]"
> "Need more resources on [area]"
