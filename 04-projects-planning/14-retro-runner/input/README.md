# Retro Runner — Input Folder

## What Goes Here

Project/sprint notes and team input.

### Input Files

| File | Purpose |
|------|---------|
| `sprint-notes.md` | Notes from the period |
| `team-input.md` | What team members said |
| `metrics.md` | Quantitative data |
| `previous-retros/` | For pattern tracking |

---

## Team Input Format

```markdown
# Team Input

## [Person Name]

### What went well
- [Item]
- [Item]

### What could improve
- [Item]
- [Item]

### Other thoughts
[Free-form]
```

---

## Previous Retros

For pattern detection:
```
input/
└── previous-retros/
    ├── sprint-45-retro.md
    └── sprint-46-retro.md
```

## Without Input Files

Works through conversation:
> "Run retro: [describe what happened during the sprint]"
