# Retro Runner — Output Folder

## What Appears Here

```
output/
├── retrospective.md    # Complete retro
├── experiments.md      # Things to try
├── patterns.md         # Recurring themes
└── team-dynamics.md    # Team observations
```

---

## Using the Outputs

### During Retro Meeting
- Use `retrospective.md` as facilitation guide
- Review `patterns.md` for chronic issues
- Commit to experiments from `experiments.md`

### After Retro
- Share final retrospective with team
- Add experiments to sprint planning
- Save for future pattern tracking
