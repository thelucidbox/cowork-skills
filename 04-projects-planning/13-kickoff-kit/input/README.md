# Kickoff Kit — Input Folder

## What Goes Here

Project briefs and template customizations.

### Project Information

| File | Purpose |
|------|---------|
| `project-brief.md` | Project description |
| `constraints.md` | Budget, timeline, team info |
| `stakeholders.md` | Known stakeholders |
| `risks.md` | Known concerns |

### Templates

| Folder | Purpose |
|--------|---------|
| `templates/` | Your org's formats |

---

## Folder Structure

```
input/
├── project-brief.md    # Required or describe verbally
├── constraints.md      # Optional
├── stakeholders.md     # Optional
└── templates/          # Optional
    └── [your templates]
```

---

## Project Brief Example

```markdown
# Project Brief

## Project Name
[Name]

## Overview
[What we're trying to do]

## Goals
- [Goal 1]
- [Goal 2]

## Known Constraints
- Budget: [Amount]
- Timeline: Must ship by [date]
- Team: [Size/composition]

## Key Stakeholders
- [Stakeholder 1]: [Interest]
- [Stakeholder 2]: [Interest]

## Known Risks/Concerns
- [Concern 1]
- [Concern 2]
```
