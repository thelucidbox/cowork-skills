# Scope Creep Detector — Output Folder

## What Appears Here

```
output/
├── scope-analysis.md        # Full comparison
├── drift-items.md           # Categorized changes
├── impact-assessment.md     # Quantified impact
└── renegotiation-points.md  # Talking points
```

---

## Using the Outputs

### For Stakeholder Discussion
- Lead with `scope-analysis.md` executive summary
- Have `drift-items.md` ready for specifics
- Use `renegotiation-points.md` as conversation guide

### For Decision Making
- Review options in renegotiation points
- Pick approach that fits constraints
- Document decision for future reference

---

## After Renegotiation

Update original-spec.md with new agreed scope to reset baseline.
