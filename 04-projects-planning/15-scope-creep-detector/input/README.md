## What Goes Here

Documents showing original and current scope.

### Required Files

| File | Purpose |
|------|---------|
| `original-spec.md` | What was originally agreed |
| `current-state.md` | What's now expected |

### Optional Files

| File | Purpose |
|------|---------|
| `change-log.md` | How scope evolved |
| `project-context.md` | Timeline, budget, constraints |

---

## Example: original-spec.md

```markdown
# Original Specification

## Date
[Date approved/signed]

## Scope

### Features
1. [Feature 1]
2. [Feature 2]

### Out of Scope
- [Explicitly excluded]

## Timeline
[Original estimate]

## Budget
[Original budget]
```

---

## Example: current-state.md

```markdown
# Current Expectations

## Date
[Today]

## What's Now Expected

### Features
1. [Feature 1] (original)
2. [Feature 2] (original)
3. [Feature 3] (added)
4. [Feature 4] (added)

## Current Timeline
[Current estimate]
```

---

## Without Files

Works conversationally:
> "Compare: Original scope was X, Y, Z. Now we're expected to deliver A, B, C, D, E."
