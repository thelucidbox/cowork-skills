# Weekly Momentum Report — Output Folder

## What Appears Here

```
output/
├── weekly-momentum.md        # Complete synthesis
├── wins.md                   # Shareable accomplishments
├── blockers.md               # Stuck items + strategies
└── next-week-priorities.md   # Focus recommendations
```

---

## File Descriptions

### weekly-momentum.md
Complete weekly report including:
- Wins (major, progress, small)
- Blockers with unblocking strategies
- Patterns observed
- Energy audit
- Next week priorities

### wins.md
Accomplishments extracted and formatted:
- Ready for brag doc
- Suitable for standup updates
- Manager-ready format

### blockers.md
Stuck items with analysis:
- How long stuck
- Impact assessment
- Suggested next move for each
- Categorized by blocker type

### next-week-priorities.md
Recommended focus areas:
- Must-do priorities
- Should-address items
- Consider-dropping suggestions
- Based on momentum and blockers

---

## Using the Outputs

### Immediately
- Review `weekly-momentum.md` for reflection
- Add `wins.md` to brag document
- Address high-impact items from `blockers.md`

### Planning
- Use `next-week-priorities.md` for weekly planning
- Block time for stuck items
- Schedule based on energy patterns

### Archiving
Save reports for trend tracking:
```
reports/
├── 2024-week-01-momentum.md
├── 2024-week-02-momentum.md
└── ...
```

### Sharing (Optional)
- Share wins with manager
- Use blockers for 1:1 discussions
- Priorities for team alignment

---

## Report Continuity

For best pattern detection:
1. Save each week's report
2. Reference previous report when generating new one
3. After 4+ weeks, ask for trend analysis
