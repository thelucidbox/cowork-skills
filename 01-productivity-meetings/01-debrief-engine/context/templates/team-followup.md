# Team Follow-up Email Template

## Structure

```
Subject: [Meeting Name] Recap + Action Items

Hey team,

Quick recap from [meeting name] on [date]:

**Decisions Made:**
- [Decision 1]
- [Decision 2]

**Action Items:**
- @[Name]: [Task] by [date]
- @[Name]: [Task] by [date]

**Open Questions:**
- [Question needing resolution]

**Next Steps:**
[What happens next / next meeting]

[Casual sign-off]
[Your name]
```

## Tone Notes

- Use "Hey team" or "Hi all" — match your usual style
- Keep bullets tight — one line each
- @ mention format for easy scanning
- Okay to be brief — team knows context

## Length Target

150-300 words. Shorter is better if nothing complex.

## Customization Points

- `[Meeting Name]`: From calendar or notes
- `[date]`: Meeting date
- `[Casual sign-off]`: "Thanks!" / "Talk soon" / "Cheers"
