# Decision Capture — Input Folder

## What Goes Here

Optional supporting documents for decision analysis.

### Useful Inputs

| File | What It Adds |
|------|--------------|
| `context.md` | Background information |
| `options-research.md` | Research on options |
| `stakeholder-input.md` | Perspectives from others |
| `previous-decisions/` | Past similar decisions |

---

## When to Use Input Files

### Use Files When
- Decision has extensive background documentation
- You've already researched options
- Multiple stakeholders have provided input
- You want to reference past decisions

### Skip Files When
- Starting from scratch
- Quick decision needed
- Prefer conversational exploration
- Sensitive decision (keep in conversation only)

---

## File Formats

Any text format works:
- `.md` (Markdown)
- `.txt` (Plain text)
- `.docx` (Word)
- `.pdf` (PDF)

---

For pattern matching across decisions:

input/

    ├── 2023-06-team-structure.md

Reference like:
