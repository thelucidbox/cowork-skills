# Downloads Archaeologist — Rules & Guidelines

## Output Format

### downloads-report.md
```markdown
# Downloads Excavation Report

## Summary
- Total files: [count]
- Date range: [oldest] to [newest]
- Total size: [size]

## Categories Found
| Category | Count | Size | Action |
|----------|-------|------|--------|
| Documents | [n] | [size] | Organize |
| Images | [n] | [size] | Organize |
| Duplicates | [n] | [size] | Delete |

## Treasures Recovered
- [Important file 1]
- [Important file 2]

## Safe to Delete
- [Junk file list]
```

---

## Category Rules

### Keep & Organize
- Documents (PDF, DOCX, XLSX)
- Images (personal photos)
- Archives (ZIP with content)
- Installers (current versions)

### Review Before Delete
- Old installers
- Duplicate files
- Partial downloads
- Temporary files

### Safe to Delete
- .tmp files
- .crdownload files
- Duplicate copies
- Empty files

---

## Identification Patterns

### Document Sources
- `invoice_` → Financial
- `receipt_` → Financial
- `report_` → Work
- `IMG_` → Photos

### Junk Patterns
- `(1)`, `(2)` → Duplicates
- `.part` → Incomplete
- `.tmp` → Temporary
