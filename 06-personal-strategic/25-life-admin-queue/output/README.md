# Life Admin Queue — Output Folder

## What Appears Here

```
output/
├── admin-queue.md           # Prioritized list
├── calendar-blocks.ics      # Time blocks
└── weekly-schedule.md       # Day-by-day plan
```

---

## Using the Outputs

### Starting Your Admin Block
1. Open `admin-queue.md`
2. Work through batched items
3. Check off completed
4. Note anything that needs follow-up

### Calendar Import
Import `calendar-blocks.ics` to your calendar.

### Weekly Planning
Review `weekly-schedule.md` on Sunday evening.

---

## Maintenance

- Update queue as new items arise
- Mark completed items
- Re-prioritize weekly
