# Annual Review Prep — Input Folder

## What Goes Here

Accomplishment evidence and context.

### Recommended Structure

```
input/
├── accomplishments-log.md    # Running log of wins
├── praise/                   # Recognition received
│   └── [kudos emails/notes]
├── goals.md                  # Year's goals
└── review-template.md        # Company format
```

---

## Accomplishments Log

Best if maintained year-round:

```markdown
# Accomplishments Log 2024

## January
- Shipped [feature]
- Led [meeting series]

## February
- Fixed [critical bug]
- Mentored [person] on [topic]

## March
...
```

---

## Praise Collection

Forward kudos to folder:
```
praise/
├── 2024-01-15-boss-kudos.md
└── 2024-03-22-client-thanks.md
```

---

## Goals Document

If you set goals at year start:
```markdown
# 2024 Goals

1. [Goal 1]: [Target metric]
2. [Goal 2]: [Target metric]
```

---

Works conversationally:
> "Help me compile my accomplishments for 2024. The big things were..."
