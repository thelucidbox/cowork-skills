# Network Mapper — Input Folder

## What Goes Here

Network configuration and manual contact data.

### Configuration

```
input/
├── network-config.md     # Tiers and preferences
└── network.md            # Manual contact list
```

---

## Tier Configuration

```markdown
# network-config.md

## Tiers

### Tier 1: Monthly
- Investors
- Key clients
- Close mentors

### Tier 2: Quarterly
- Industry peers
- Former colleagues

### Tier 3: Semi-annual
- Conference contacts
- Extended network
```

---
