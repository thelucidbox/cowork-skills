# Network Mapper — Output Folder

## What Appears Here

```
output/
├── network-health.md          # Dashboard
├── reconnection-drafts/       # Outreach messages
└── network-visualization.md   # Structure map
```

---

## Using the Outputs

### Quarterly Review
1. Check `network-health.md`
2. Prioritize at-risk relationships
3. Send reconnection messages
4. Update contact notes

### Before Events
- Review who might attend
- Prepare reconnection outreach
- Identify intro opportunities

### Weekly Maintenance
- Send 1-2 reconnection messages
- Note recent interactions

---

## Reconnection Best Practices

1. Review draft, don't send verbatim
2. Add personal touches
3. Be genuine, not transactional
4. Follow up if no response
5. Update notes after reconnecting
