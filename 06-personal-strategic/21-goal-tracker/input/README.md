# Goal Tracker — Input Folder

## What Goes Here

Goals and progress notes.

### Structure

```
input/
├── goals.md              # Your goals
└── progress/
    ├── week-01.md        # Weekly notes
    ├── week-02.md
    └── ...
```

---

## Goals Format

```markdown
# goals.md

## Goal 1: [Name]
- **Target**: [Measurable outcome]
- **Due**: [Date]
- **Priority**: High
- **Key Results**:
  - [KR1]
  - [KR2]

## Goal 2: [Name]
...
```

---

## Progress Format

```markdown
# Week 01

## Goal 1
- [Activity]
- [Progress made]

## Goal 2
- [Activity]
```

---

## Without Structured Files

Works conversationally:
> "My goals are X, Y, Z. This week I did A, B, C."
