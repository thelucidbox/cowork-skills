## No Setup Required

Goal Tracker analyzes files you provide. No external integrations needed.

---

## How It Works

### Provide Two Things
1. **Goals document**: What you're trying to achieve
2. **Progress evidence**: Notes, task logs, activity records

### Analysis Generated
- Current status per goal
- Trajectory (improving/declining)
- Recommendations

## Input Structure

### Goals Document Format
```markdown
# goals.md

## Q1 2024 Goals

### Goal 1: [Name]
- **Target**: [Specific metric]
- **Due**: [Date]
- **Priority**: High/Medium/Low

### Goal 2: [Name]
...
```

### Progress Notes Format
```markdown
# progress/

## Week 1
- [Activity related to Goal 1]
- [Progress on Goal 2]

## Week 2
...
```
