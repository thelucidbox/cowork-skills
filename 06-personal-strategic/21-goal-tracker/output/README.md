# Goal Tracker — Output Folder

## What Appears Here

```
output/
├── goal-dashboard.md     # Visual status
```

## Using the Outputs

### Weekly Review
- Check dashboard status
- Note trajectory changes
- Pick top actions

### When Stuck
- Review recommendations
- Choose rescue plan option
- Consider dropping if appropriate

### Quarterly Planning
- Archive current dashboard
- Set new goals
- Learn from patterns
