# Energy Audit — Input Folder

## What Goes Here

Energy tracking data and notes.

### Structure

```
input/
├── energy-log.md         # Daily energy ratings
└── notes/                # Optional: detailed notes
```

---

## Energy Log Format

```markdown
# Energy Log

## 2024-01-15
- Energy: 7/10
- Sleep: Good
- Exercise: Morning run
- Notes: Productive day

## 2024-01-16
- Energy: 5/10
- Notes: Back-to-back meetings all day
```

---

## Simple Format

Even simpler:
```
01-15: 7
01-16: 5
01-17: 8
```

---

## Without Input Files

Works with calendar alone:
> "Analyze my calendar for energy patterns"

Or conversationally:
> "I usually feel drained after all-day meetings"
