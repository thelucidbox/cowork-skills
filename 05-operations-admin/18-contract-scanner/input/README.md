# Contract Scanner — Input Folder

## What Goes Here

Contract documents to analyze.

### Supported Formats

- `.pdf`
- `.docx`
- `.doc`
- `.txt` / `.md`

---

## Structure

```
input/
├── contract.pdf              # Contract to analyze
├── attachments/              # Any exhibits/schedules
│   └── schedule-a.pdf
└── scan-config.md           # Optional preferences
```

---

## Best Practices

### Include Everything
- Main agreement
- All schedules and exhibits
- Amendments
- Order forms

### Note Your Role
> "I'm the customer in this agreement"

### Flag Concerns
> "Particularly concerned about liability"

---

## Without Input Files

Paste contract text directly:
> "Analyze this contract: [paste text]"
