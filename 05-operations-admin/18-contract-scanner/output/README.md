# Contract Scanner — Output Folder

## What Appears Here

```
output/
├── contract-summary.md      # Key terms
├── red-flags.md             # Potential concerns
├── questions-for-legal.md   # For your lawyer
└── calendar-entry.ics       # Important dates
```

---

## Using the Outputs

### For Quick Understanding
Review `contract-summary.md` for key terms.

### For Legal Review
1. Share `red-flags.md` with lawyer
2. Discuss `questions-for-legal.md`
3. Get professional advice

### For Calendar
Import `calendar-entry.ics` for:
- Contract end date
- Renewal notice deadline
- Payment milestones

---

## Important Reminder

**ALWAYS have a lawyer review before signing.**

These outputs help you understand and prepare questions—they are not a substitute for legal advice.
