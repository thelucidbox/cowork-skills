# File Archaeologist — Output Folder

## What Appears Here

```
output/
├── organized-folder/        # Reorganized files
├── manifest.md              # Complete index
├── delete-candidates.md     # Suggested deletions
└── rediscovered-items.md    # Worth revisiting
```

---

## Using the Outputs

### Before Applying Changes
1. Review `manifest.md` for planned changes
2. Check `delete-candidates.md` carefully
3. Look at `rediscovered-items.md` for surprises

### Applying Changes
1. Confirm manifest looks correct
2. Move/rename files
3. Keep manifest for undo reference

### Recovery
If something goes wrong:
1. Reference manifest for original locations
2. Move files back
3. Rename to original names

---

## Workflow Recommendation

1. **Analyze first**: Run in read-only mode
2. **Review outputs**: Check all suggestions
3. **Apply selectively**: Make changes you agree with
4. **Keep manifest**: Your safety net
