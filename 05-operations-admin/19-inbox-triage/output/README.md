# Inbox Triage — Output Folder

## What Appears Here

```
output/
├── inbox-triage.md          # Prioritized list
├── response-drafts/
│   └── [subject].md         # Draft responses
└── ignore-list.md           # Safe to archive
```

---

## Using the Outputs

### Working Through Triage
1. Handle **Urgent** first
2. Use drafts as starting points
3. Schedule time for **This Week**
4. Batch archive from **ignore-list.md**

### Response Drafts
- Review and customize before sending
- Alternative drafts included
- Match or adjust tone as needed

### Archive List
- Review before bulk archiving
- May include false positives
- Better to keep than miss important

---

## Daily Workflow

1. Morning: Run triage
2. Review urgent items
3. Handle top priorities
4. End of day: Quick triage check
