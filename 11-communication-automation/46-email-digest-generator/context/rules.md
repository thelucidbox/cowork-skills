## Priority Rules

### High Priority
- Direct requests for action
- VIP senders
- Deadline mentioned
- Escalation language

### Medium Priority
- Updates requiring awareness
- Questions that need answers
- Project-related

### Low Priority
- FYI/CC emails
- Newsletters
- Automated notifications
- Marketing

---

## Summary Guidelines

- One sentence per email max
- Focus on action/decision needed
- Include deadline if mentioned
- Note if reply expected
