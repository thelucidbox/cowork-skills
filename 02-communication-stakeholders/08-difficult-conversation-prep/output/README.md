# Difficult Conversation Prep — Output Folder

## What Appears Here

```
output/
├── conversation-prep.md        # Complete preparation
├── talking-points.md           # Key messages
├── anticipated-responses.md    # Their reactions + rebuttals
└── de-escalation-options.md    # If things heat up
```

---

## File Descriptions

### conversation-prep.md
Complete preparation including:
- Situation summary
- Your position and BATNA
- Talking points overview
- Success criteria

### talking-points.md
What to say:
- Opening frame
- Key messages with scripts
- Phrases to use and avoid

### anticipated-responses.md
Their likely reactions:
- What they might say
- Why they'd say it
- Your prepared response

### de-escalation-options.md
If things get difficult:
- Prevention tactics
- Cooling phrases
- Strategic pause options
- How to abort safely

---

## Using the Outputs

### Before the Conversation
1. Read through `conversation-prep.md`
2. Practice `talking-points.md` out loud
3. Review `anticipated-responses.md`
4. Familiarize with `de-escalation-options.md`

### During (If Needed)
- Keep `de-escalation-options.md` nearby
- Reference talking points if you blank

### After
- Consider deleting sensitive files
- Note what worked for future reference

---

## Privacy Recommendation

Difficult conversation prep often contains sensitive content. Consider:
- Not saving files (keep in conversation)
- Deleting after use
- Storing in secure location only
