# Difficult Conversation Prep — Input Folder

## What Goes Here

Optional context files to enhance your preparation.

### Useful Inputs

| File | Purpose |
|------|---------|
| `situation.md` | Detailed situation description |
| `their-perspective.md` | What you know about their view |
| `history.md` | Relevant relationship history |
| `constraints.md` | What you can/can't offer |

---

## Folder Structure

```
input/
├── situation.md          # The issue in detail
├── their-perspective.md  # Their likely viewpoint
├── history.md            # Past relevant interactions
└── constraints.md        # Your boundaries
```

---

## When to Use Files

### Use Files When
- Situation is complex
- You want to capture details before conversation
- Multiple attempts at this conversation
- You're preparing in advance

### Skip Files When
- Quick prep needed
- Prefer conversational exploration
- Sensitive and shouldn't be written down

---

## File Formats

### situation.md Example
```markdown
## The Situation

[Person] and I need to discuss [topic].

The issue is: [core problem]

This matters because: [stakes]

We've tried before: [prior attempts, or "first time"]
```

### their-perspective.md Example
```markdown
## Their Likely View

They probably think: [their perspective]

They want: [their goals]

They're concerned about: [their fears]

What I know for sure: [verified]
What I'm guessing: [assumptions]
```
