# Relationship Pulse — Output Folder

## What Appears Here

```
output/
├── profiles/
│   ├── sarah-chen.md
│   └── [name].md
├── relationship-health.md
├── reconnection-drafts/
│   └── [name]-reconnect.md
└── insights.md
```

---

## File Descriptions

### profiles/{name}.md
Individual relationship files:
- Contact information
- Interaction history
- Notes and observations
- Conversation starters

### relationship-health.md
Dashboard view:
- Health summary by tier
- Needs attention list
- At-risk relationships
- Network statistics

### reconnection-drafts/
Ready-to-customize outreach:
- For dormant relationships
- Personalized based on profile
- Warm, non-apologetic tone

### insights.md
Network patterns:
- Communication trends
- Relationship distribution
- Recommendations

---

## Using the Outputs

### Regular Review (Weekly)
1. Check `relationship-health.md` for at-risk
2. Review any VIPs needing attention
3. Customize and send reconnection drafts

### Pre-Meeting
1. Review relevant profile
2. Note conversation starters
3. Update after meeting

### Relationship Building
1. Use insights to improve patterns
2. Adjust tiers as relationships evolve
3. Add notes after meaningful interactions

---

## Accumulation Over Time

Profiles improve with use:
- More interactions = richer history
- More notes = better context
- Longer tracking = clearer patterns

Save outputs to build relationship intelligence.
