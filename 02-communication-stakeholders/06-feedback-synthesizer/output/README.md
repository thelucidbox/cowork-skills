# Feedback Synthesizer — Output Folder

## What Appears Here

```
output/
├── feedback-synthesis.md      # Complete analysis
├── quotes-by-theme.md         # Organized quotes
├── priority-improvements.md   # Ranked actions
└── response-drafts/
    └── [responses as needed]
```

---

## File Descriptions

### feedback-synthesis.md
Complete analysis including:
- Executive summary
- Key themes with frequency
- Sentiment breakdown
- Pattern analysis
- Priority improvements overview

### quotes-by-theme.md
Original feedback preserved:
- Quotes organized by theme
- Positive/negative separation
- Attribution when available

### priority-improvements.md
Actionable output:
- Ranked improvements
- Impact and effort notes
- Source frequency
- Suggested actions

### response-drafts/
Individual response templates:
- For feedback requiring response
- Customizable before sending
- Tone-appropriate

---

## Using the Outputs

### Sharing Results
- `feedback-synthesis.md`: Full stakeholder report
- `priority-improvements.md`: Action planning
- `quotes-by-theme.md`: Support for recommendations

### Taking Action
1. Review priorities against your constraints
2. Pick top 1-3 improvements to address
3. Respond to feedback requiring response
4. Share synthesis with team
