# Photo Optimizer — Output Folder

## What Appears Here

```
output/
├── optimized/
│   └── [processed images]
└── optimization-report.md
```

---

## Using the Outputs

### Upload to Web
Use images from `optimized/` folder.

### Review Quality
Check `optimization-report.md` for before/after.

### Adjust Settings
If quality isn't right, re-run with different settings.

---

## Multiple Sizes

For responsive images, request:
> "Generate thumb, medium, and large versions"

Output:
```
optimized/
├── thumb/
├── medium/
└── large/
```
