# Photo Optimizer — Input Folder

## What Goes Here

Images to optimize.

### Structure

```
input/
├── photos/
│   └── [your images]
└── optimizer-config.md
```

---

## Supported Formats

- `.jpg` / `.jpeg`
- `.png`
- `.gif`
- `.webp`
- `.heic`

---

## Configuration

```markdown
# optimizer-config.md

## Settings

### Target
- Max width: 1200
- Quality: 85
- Format: webp

### Keep Originals
- Yes
```
