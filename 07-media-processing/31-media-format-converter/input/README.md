# Media Format Converter — Input Folder

## What Goes Here

Media files to convert.

### Structure

```
input/
├── videos/
├── audio/
└── images/
```

---

## Supported Formats

### Video
`.mov`, `.mp4`, `.avi`, `.mkv`, `.webm`

### Audio
`.wav`, `.mp3`, `.flac`, `.m4a`, `.aac`

### Images
`.png`, `.jpg`, `.heic`, `.webp`
