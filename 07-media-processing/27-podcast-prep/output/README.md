# Podcast Prep — Output Folder

## What Appears Here

```
output/
├── podcast-ready.mp3      # Final processed audio
├── transcript.md          # Optional transcript
└── processing-log.md      # Processing details
```

---

## File Descriptions

### podcast-ready.mp3
Final processed audio:
- Normalized loudness
- Noise reduced
- Silences adjusted
- Intro/outro added
- Ready to publish

### transcript.md
Text transcript (if requested):
- Timecoded
- Speaker labels (if identified)
- Searchable text

### processing-log.md
What was done:
- Source analysis
- Processing steps
- Technical specs
- Commands used

---

## Quality Check

Before publishing:
1. Listen on headphones
2. Check on phone speaker
3. Verify intro/outro transitions
4. Confirm loudness is consistent
