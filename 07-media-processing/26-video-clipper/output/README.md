# Video Clipper — Output Folder

## What Appears Here

```
output/
├── clips/
│   ├── clip-001.mp4
│   └── clip-002.mp4
├── clip-manifest.md
└── thumbnails/
    └── clip-001.jpg
```

---

## File Descriptions

### clips/
Extracted video segments:
- Quality matched to source
- Named descriptively
- Ready for use

### clip-manifest.md
Index of all clips:
- Timestamps
- Descriptions
- Transcripts
- ffmpeg commands used

### thumbnails/
Preview images:
- One per clip
- Representative frame

---

## Using the Clips

### For Social Media
- Check platform requirements
- May need format conversion
- Thumbnails for preview

### For Presentations
- Import directly
- Reference manifest for context

### For Archive
- Save manifest for future reference
- Keep originals

---

## Re-Extraction

If clip needs adjustment:
- Note "5 seconds earlier" etc.
- Or provide exact timestamps
- Re-run with new parameters
