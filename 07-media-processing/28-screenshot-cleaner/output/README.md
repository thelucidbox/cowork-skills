# Screenshot Cleaner — Output Folder

## What Appears Here

```
output/
├── organized-screenshots/
│   ├── work/
│   ├── personal/
│   └── reference/
├── duplicates/
├── duplicates-removed.md
└── index.md
```

---

## Using the Outputs

### Browse Organized
Navigate `organized-screenshots/` by category.

### Search
Use `index.md` to find screenshots by content.

### Clean Up Duplicates
1. Review `duplicates-removed.md`
2. Check `duplicates/` folder
3. Delete when confident

---

## Maintaining Organization

As you take new screenshots:
1. Move to input folder
2. Re-run cleaner
3. Index updates automatically
