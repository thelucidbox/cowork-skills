# Screenshot Cleaner — Input Folder

## What Goes Here

Screenshots to organize.

### Direct Method
Point to existing folder:
> "Clean up ~/Desktop/Screenshots"

### Copy Method
```
input/
└── screenshots/
    └── [your screenshots]
```

---

## Supported Formats

- `.png`
- `.jpg` / `.jpeg`
- `.gif`
- `.heic`
- `.webp`

---

## Configuration

```markdown
# cleaner-config.md

## Categories
- work/
- personal/
- reference/

## Naming
- Format: YYYY-MM-DD-description
- Lowercase: Yes
```
