# Video Transcriber — Input Folder

## What Goes Here

Videos to transcribe.

### Structure

```
input/
├── videos/
│   └── [your videos]
└── transcribe-config.md
```

---

## Supported Formats

- `.mp4`
- `.mov`
- `.avi`
- `.mkv`
- `.webm`

---

## Configuration

```markdown
# transcribe-config.md

## Settings
- Output format: md, srt
- Timestamps: Every 30 seconds

## Speakers
- Speaker 1: Sarah (host)
- Speaker 2: Mike (guest)
```
