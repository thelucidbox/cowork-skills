# Video Transcriber — Output Folder

## What Appears Here

```
output/
├── transcripts/
│   └── [video-name].md
├── subtitles/
│   └── [video-name].srt
└── transcript-index.md
```

---

## Using the Outputs

### Transcripts
- Search and reference
- Use for show notes
- Create written content

### Subtitles
- Import to video editor
- Upload to video platform
- Accessibility compliance

### Index
- Search across all videos
- Find specific topics
- Reference for content

---

## Review Process

1. Read through transcript
2. Fix obvious errors
3. Verify names and terms
4. Check subtitle timing
